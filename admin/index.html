⚙️ YouTube Setup

To post directly to YouTube, you need a Google Cloud project with the YouTube Data API enabled.

  1. Go to console.cloud.google.com
  2. Create project → Enable YouTube Data API v3
  3. Create OAuth 2.0 credentials (Web App)
  4. Add redirect URI:
    https://jose-andres-pokemon.onrender.com/api/social/youtube/callback
  5. Add YT_CLIENT_ID + YT_CLIENT_SECRET to Render env vars
  6. Click Connect YouTube below
